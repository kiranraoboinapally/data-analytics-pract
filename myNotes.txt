
Installation of Node.js and git on rhel10!
Step1:
curl -fsSL https://rpm.nodesource.com/setup_20.x | sudo bash -
Step2:
sudo dnf install -y nodejs

for updating all the existing packages
sudo dnf update -y

for installing git
sudo dnf install git -y

for chckeing what are installed
dnf list installed

for searching only that exists or not!?
dnf search git

for knowing versions
dnf info git
